# 🧠 Two-Step Thinker (GigaChat Edition)

Простая и эффективная демонстрация оркестрации LLM с использованием **LangChain (LCEL)** и **Sber GigaChat**. 

Этот проект показывает, как превратить вывод одного действия нейросети во входные данные для следующего, создавая осмысленный и глубокий диалог.

## 🚀 Как это работает?

Цепочка (Chain) состоит из двух этапов:
1.  **The Explainer**: Кратко объясняет заданную тему простыми словами.
2.  **The Questioner**: Берет это объяснение и на его основе генерирует 3 провокационных вопроса для дальнейшего изучения.

Все это связано воедино с помощью **LCEL (LangChain Expression Language)**, что позволяет легко управлять потоком данных.

## 🛠 Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url-вашего-репозитория>
   cd Orchestrator
   ```

2. **Создайте виртуальное окружение и активируйте его**:
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # macOS/Linux:
   source venv/bin/activate
   ```

3. **Установите зависимости**:
   ```bash
   pip install langchain-gigachat python-dotenv langchain-community
   ```

## 🔑 Настройка (Secrets)

Создайте файл `.env` в корневой директории и добавьте туда свои ключи:

```env
# Авторизационные данные GigaChat (Base64)
GIGACHAT_CREDENTIALS="ваш_Authorization_Key"

# (Опционально) Для визуализации в LangSmith
LANGCHAIN_TRACING_V2=true
LANGCHAIN_API_KEY="ваш_LangSmith_ключ"
LANGCHAIN_PROJECT="Two-Step-Thinker-GigaChat"
```

> [!IMPORTANT]
> Файл `.env` уже добавлен в `.gitignore`, поэтому ваши ключи никогда не попадут в публичный доступ.

## 🏃 Запуск

Просто запустите скрипт и введите интересующую вас тему:

```bash
python two_step_thinker.py
```

## ✨ Фишки проекта
- **Динамический ввод**: Вы можете задавать любые темы прямо в консоли.
- **Безопасность**: Использование переменных окружения для ключей.
- **Трассировка**: Интеграция с LangSmith для визуализации каждого шага "размышления" модели.
- **Sber GigaChat**: Полная поддержка российских LLM.
